# LapTrack_F1
LapTrack F1 é um dashboard em Python utilizando Dash, Plotly e FastF1, projetado para analisar telemetria e desempenho de pilotos de Fórmula 1. Ele permite visualizar tempos de volta, posições no circuito, comparações entre pilotos e estatísticas de corrida de forma dinâmica e visual.
